# Sistema de Gestion de Biblioteca Personal

## Descripcion
Sistema completo de gestion de biblioteca desarrollado en Python con Tkinter y MySQL. Implementa operaciones CRUD y sigue principios de Programacion Orientada a Objetos (POO) para el taller de evaluacion.

## Caracteristicas
- Gestion completa de libros (CRUD)
- Gestion de usuarios y autores
- Control de prestamos y devoluciones
- Interfaz grafica con Tkinter
- Base de datos MySQL
- Programacion Orientada a Objetos

## Tecnologias Utilizadas
- Python 
- Tkinter 
- MySQL
